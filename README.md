# java-web-maven-vaadin-secure-gzip-compress-rsa-encrypted-argon2-encoded

## Description
A vaadin login example using
RSA encryption with argon2
one-way hash.

## Tech stack
- java
- maven
  - gwt
  - rsa
  - argon2
  - vaadin

## Docker stack
- adoptopenjdk/maven-openjdk11

## To run
`sudo ./install.sh -u`
http://localhost
- username: user
- password: pass

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
